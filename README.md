# ECC-Cpp
Elliptic curve cryptographie in c++
Implements Point multiplication for the field $F_{47^{46}}$  
In Main yu have a point, which you can multiply bi some scalar.
The scalar is a array of lenght 8 with long long unsigned int, where the scalar is the number in hex, when you concatenate the elements. which this scalars up to $2^{256}-1$ is possible.
